# memcache-web
一个**非常**简单memcache的gui界面，后台由node编写

## install
需装memcache并启动，各系统安装说明  
[linux memcache][1]  
[window memcache][2]  
```
git clone https://github.com/jayZOU/memcache-web.git
cd memcache-web
npm i
npm start

http://localhost:12306/index.html
```
